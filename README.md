This is the example app for my [YouTube video](https://youtu.be/l1AkkVEa4ZM) talking about Vuex 5 and Vite. 

If you want to try it out:

1. `yarn install`
2. `node src/server.js` for the back-end
3. `yarn vite` for the front-end
